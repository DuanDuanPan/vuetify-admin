# vuetify-admin
一个vuetify后台基础模板
# 预览地址
<a href="http://fhluo.gitee.io/vuetify-admin" target="_blank">proview</a>
# 使用步骤
1.克隆项目到本地<br/>
`git clone https://github.com/Kirito001/vuetify-admin.git`<br/>
2.进入刚下载的项目目录<br/>
`cd vuetify-admin`<br/>
3.安装依赖<br/>
`npm install`<br/>
4.运行项目<br/>
`npm run serve`<br/>
